News Management Demo App

Purpose of the project

This is a demo app for the News Management System.

In this project, the words "items" and "news" are considered equal. Developers can use these two words interchangeably.

Main function

  • List, Create, Update, Delete News (Items)
  • List, Create, Update, Delete News Category
  • Search, Sort, Filter Items by Status
  • Search, Sort Categories

Technical Stack

  • FE Frameword: Angular
  • UI Libraries: Tailwind, Material Angular
  • Mockup Server: json-server
  • RTE: TinyMCE
